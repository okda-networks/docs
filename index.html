ONM Device Manager

ONM Device Manager is a sweet of tools that allows easy managment of any Linux based system, it offers an intutive user CLI and GUI for configuring and monitoring the system.

Okda Network Manager

ONM centralized managment platform that allows you to monitor and configure your network nodes from a single pane of glass.

Okda Network Controller

ONC is a platform that maintains a realtime view of the network nodes states and allows creating event driven network control plan applications.
